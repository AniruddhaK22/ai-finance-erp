# AI Finance ERP

## Project Overview

AI Finance ERP is a comprehensive enterprise resource planning solution designed for financial management using advanced technologies such as AI, blockchain and automated invoice processing. The platform integrates various functionalities such as accounting, invoicing, reporting, and customer management to streamline financial operations for small to medium-sized enterprises.

### Features
- **User Authentication**: Secure user authentication with JWT.
- **Invoice Parsing**: Automate invoice processing with AI algorithms.
- **Dashboard**: User-friendly dashboard for insights and analytics.
- **Data Storage**: Secure data storage using MongoDB.

### Technologies Used
- **Backend**: Node.js, Express.js, MongoDB
- **Frontend**: React, Tailwind CSS
- **Deployment**: Docker
